# 159. Longest Substring with At Most Two Distinct Characters

> 159. 至多包含两个不同字符的最长子串

给你一个字符串 s ，请你找出 至多 包含 两个不同字符 的最长子串，并返回该子串的长度。

### Example 1:

```
输入：s = "eceba"
输出：3
解释：满足题目要求的子串是 "ece" ，长度为 3 。

```

### Example 2:

```
输入：s = "ccaabbb"
输出：5
解释：满足题目要求的子串是 "aabbb" ，长度为 5 。

```

### Constraints:

* 1 <= s.length <= 105
* s 由英文字母组成

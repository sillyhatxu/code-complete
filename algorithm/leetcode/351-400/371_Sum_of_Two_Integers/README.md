# 371. Sum of Two Integers

Given two integers a and b, return the sum of the two integers without using the operators + and -.

### Example 1:

```
Input: a = 1, b = 2
Output: 3
```

### Example 2:

```
Input: a = 2, b = 3
Output: 5
```

### Constraints:

* -1000 <= a, b <= 1000

### Translate:

> 371. 两整数之和

给你两个整数 `a` 和 `b` ，**不使用** 运算符 `+` 和 `-` ，计算并返回两整数之和。
